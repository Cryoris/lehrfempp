Detailed Description

template<typename SCALAR, typename FUNCTOR>
class lf::fe::LinearFELocalLoadVector< SCALAR, FUNCTOR >

Class for computation of local load vector for linear finite elements.

Template Parameters
FUNCTORobject with an evaluation operator of signature std::function<double(const Eigen::Vector2d &)>, which supplies the source function

Computation is based on vertex based quadrature

Note
The element vector returned by the Eval() method will always have length 4 also for triangles.

This class complies with the requirements for the template parameter ELEM_VEC_COMP of the function AssembleVectorLocally().
