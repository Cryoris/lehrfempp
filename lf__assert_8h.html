LF_VERIFY_MSG(expr, msg) aborts execution of the code if expr evaluates to false.

At the moment this macro just forwards everything to BOOST_VERIFY_MSG() and throws an exception if the condition is not satisfied in order to help IDE tools detect that BOOST_VERIFY_MSG(false, "message") always aborts execution.
