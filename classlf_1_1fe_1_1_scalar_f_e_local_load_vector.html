Detailed Description

template<typename SCALAR, typename FUNCTOR>
class lf::fe::ScalarFELocalLoadVector< SCALAR, FUNCTOR >

Local computation of general element vector for scalar finite elements.

Template Parameters
SCALARunderlying scalar type, usually double or complex<double>
FUNCTORobject with an evaluation operator of signature std::function<double(const Eigen::Vector2d &)>, which supplies the source function

Computation is based on a quadrature rules supplied by the LehrFEM++ lf::quad::QuadRule module.

This class complies with the requirements for the template parameter ELEM_VEC_COMP of the function AssembleVectorLocally().
